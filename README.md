# 🌐 LinkList

私の各種のリンクをシンプルに整理したポートフォリオサイトです。
お気に入りのサイトやツールをまとめ、すぐにアクセスできるようにしています。

---

## 🧭 概要

**LinkList** は、静的 HTML・CSS・JavaScript で構築されたシンプルなリンク集です。

---

## 🚀 主な機能

- 🔗 各種SNSリンクの表示
- 🎨 シンプルで見やすい UI
- 📱 スマホ・タブレット対応のレスポンシブデザイン
- 🌈 ダークモード対応

---

## 🏗 使用技術

| 技術 | 用途 |
|------|------|
| HTML5 | ページ構造 |
| CSS3 | レイアウトとスタイル |
| JavaScript (Vanilla JS) | UIアニメーション |
| GitHub Pages | 公開・ホスティング |
